# PlayerSwapper
プレイヤーの位置を入れ替えるデータパック

> **対応バージョン**

**1.17～**

> **動作確認バージョン**

**1.19.x**


# ダウンロード方法
右側の[Releases]をクリックしMaki-PlayerSwapper.zipをダウンロードするだけ！

# データパックの導入方法
Maki-PlayerSwapper.zipを解凍し中にあるデータパックをworldフォルダのdatapacksに入れる

# データパックの使用方法
```mcfunction
# 設定本を入手する
/loot give @s loot swapper:book
# ゲームをリセットする
/function swapper:init
``
